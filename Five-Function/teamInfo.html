Team. Five Function

팀 소개 :

우리 팀은 각자 다른 개성과 강점을 가진 5개의 함수들로 이루어진 하나의 객체입니다.
아직은 알에서 갓 깨어난 병아리처럼 서툴고 작은 발걸음이지만, 서로 연결된 함수들이 협력하며 꾸준히 성장하고, 새로운 데이터를 배우며 강력한 객체로 거듭날 것입니다.

우리가 가진 함수

  • 열정으로_배우기();
  • 함께_고민하기();
  • 꾸준히_성장하기();
  • 긍정적으로_협력하기();
  • 끝까지_완주하기();
